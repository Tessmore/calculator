# README

Create a minimal React app that makes use of TypeScript, Vite, Prettier et cetera.

## Available commands

Run in development mode

```bash
  yarn dev
```

Create production build

```bash
  yarn build
```

Run ESLint linting

```bash
  yarn lint
```

Run Prettier formatting

```bash
  yarn format
```

Deploy using GitHub pages

```bash
  yarn run deploy
```
